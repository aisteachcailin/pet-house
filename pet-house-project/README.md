# ПЭТ-Хаус НН - Проект сайта

Проект построен на **Vite** для быстрой разработки и сборки.

## Структура проекта

```
pet-house-project/
├── dist/              # Собранные файлы для продакшена
├── public/           # Статические файлы (изображения)
│   └── images/
├── src/              # Исходные файлы
│   ├── components/   # HTML компоненты
│   ├── js/           # JavaScript файлы
│   ├── pages/        # Страницы сайта
│   └── styles/       # SCSS стили
│       ├── scss/     # Исходные SCSS файлы
│       └── css/      # Скомпилированные CSS (автоматически)
├── index.html        # Главная страница (в корне для Vite)
├── vite.config.js    # Конфигурация Vite
└── package.json
```

## Установка

```bash
npm install
```

## Команды

### Разработка
```bash
npm run dev
```
Запускает Vite dev server на `http://localhost:3000` с автоматической перезагрузкой и компиляцией SCSS.

### Сборка
```bash
npm run build
```
Собирает проект в папку `dist` с оптимизацией и минификацией.

### Просмотр собранной версии
```bash
npm run preview
```
Запускает preview сервер для проверки собранной версии из `dist`.

## Особенности

- **Vite** - быстрый dev server и сборщик
- **SCSS** - поддержка препроцессора из коробки
- **Автоматическая перезагрузка** - изменения применяются мгновенно
- **Оптимизация** - автоматическая оптимизация при сборке

## Пути к файлам

В проекте используются абсолютные пути от корня:
- CSS/SCSS: `/src/styles/scss/main.scss`
- JS: `/src/js/main.js`
- Компоненты: `/src/components/header.html`
- Изображения: `/images/hero-bg.jpg` (из папки `public/`)

## Важно

- `index.html` должен находиться в корне проекта (требование Vite)
- Статические файлы (изображения) размещаются в `public/`
- Vite автоматически обрабатывает импорты SCSS в HTML
